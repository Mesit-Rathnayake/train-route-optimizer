# Train Route Optimizer

**Train Route Optimizer** is a Java project that calculates the shortest travel time between train stations using Dijkstra’s algorithm. It demonstrates graph theory concepts and clean modular Java project structure.

## Features
- Modular architecture: `model`, `graph`, `service`, `data`
- `Station` and `Connection` classes represent the network
- `Graph` class stores station connections
- `DijkstraService` computes shortest travel times
- `TrainDataLoader` provides sample train network
- Easily extendable for more stations or real-world datasets
